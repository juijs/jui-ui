Button

Default

Button provides a total of 4 styles including the basic type. The user can designate the button style when the button is pressed.

Button Class Description
Default btn Basic button (2 tones)
Gray btn btn-gray Gray button (4 tones)
Gray-Purple btn btn-gray-purple Gray button, changed to purple when clicked (4 tones)
Gray-Black btn btn-gray-black Gray button, changed to black when clicked (4 tones)
Disabled btn btn-gray disabled Gray button, disabled
Purple btn btn-purple Purple button (4 tones)
Purple-Gray btn btn-purple-gray Purple button, changed to gray when clicked (4 tones)
Purple-Black btn btn-purple-black Purple button, changed to black when clicked (4 tones)
Disabled btn btn-purple disabled Purple button, disabled
Black btn btn-black Black button (4 tones)
Black-Purple btn btn-black-purple Black button, changed to purple when clicked (4 tones)
Black-Gray btn btn-black-gray Black button, changed to gray when clicked (4 tones)
Disabled btn btn-black disabled Black button, disabled
Gray btn btn-gray btn-flat btn-rect Gray button, rectangular form
Gray-Purple btn btn-gray-purple btn-flat btn-rect Gray button, changed to purple when clicked/rectangular form (2 tones)
Gray-Black btn btn-gray-black btn-flat btn-rect Gray button, changed to black when clicked/rectangular form (2 tones)
Disabled btn btn-gray btn-flat btn-rect disabled Gray button, disabled/rectangular form
Purple btn btn-gray btn-flat btn-rect Purple button, rectangular form (2 tones)
Purple-Gray btn btn-purple-gray btn-flat btn-rect Purple button, changed to gray when clicked/rectangular form (2 tones)
Purple-Black btn btn-purple-black btn-flat btn-rect Purple button, changed to black when clicked/rectangular form (2 tones)
Disabled btn btn-purple btn-flat btn-rect disabled Purple button, disabled/rectangular form
Black btn btn-black btn-flat btn-rect Black button, rectangular form (2 tones)
Black-Purple btn btn-black-purple btn-flat btn-rect Black button, changed to purple when clicked/rectangular form (2 tones)
Black-Gray btn btn-black-gray btn-flat btn-rect Black button, changed to gray when clicked/rectangular form (2 tones)
Disabled btn btn-black btn-flat btn-rect disabled Black button, disabled/rectangular form
